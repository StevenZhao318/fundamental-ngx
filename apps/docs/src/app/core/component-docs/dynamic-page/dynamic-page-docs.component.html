 Dynamic height of parent element

By default, Dynamic Dage component automatically calculates amount of free space it can take, and recalculates it on window resize.

If you plan on using Dynamic Page component inside container with dynamic content height (e.g. some elements inside container can appear after Dynamic Page component has been initiated), you need to apply fd-dynamic-page-wrapper directive to that container in order for Dynamic Page component to listen container's size changes instead of window size changes.
